Visual Editor Quick Guide

How to log in, edit in canvas, and publish with confidence.

Access

1) Getting to the dashboard

  1. In your browser, go to https://Hightechheroes.ai (landing page).
  2. In the address bar, append /login so it becomes https://Hightechheroes.ai/login.
  3. Press Enter to open the Bugence Visual Editor login page.
Login

Canvas

3) Editing pages in canvas mode

  1. Open the Library: From the dashboard top bar or left sidebar, click Library. If searching, choose Content/Library to view page cards.
  2. Pick a page card: Find your page, click its Edit action to open the right sidebar.
  3. Launch canvas: In the sidebar, click Edit in Canvas (top). The page opens with the overlay.
  4. Work in canvas: Hover blocks to see an outline, then double-click to open the editor popup.
  5. Locked areas: Navbar/sidebar are locked (view-only).
  6. Text: Edit content and links, then save and publish.
  7. Images: Upload/replace, set alt text, save, then publish.
Text editor

Canvas text editor popup

  • Rich text: bold, italic, headings, lists, links, undo/redo.
  • Meta bar: word count, characters, estimated read time.
  • Revert: back to last saved or previous revision (if available).
  • Save: stores a draft; publish to push live.
Image editor

Canvas image editor popup

  • Upload/replace with live preview (type/size validation).
  • Alt text with quality meter.
  • Revert to previous image/alt in one click.
  • Save: stores a draft; publish to push live.
Password reset

4) Resetting your password (OTP-based)

  1. On login, click Forgot password.
  2. Enter pete@petedulcamara.com.
  3. Receive a one-time OTP (valid 5 minutes) at that email.
  4. Enter OTP → go to Change Password.
  5. Set and confirm a new password, then log in.

If you forget after changing it in Settings: use the same Forgot Password flow; request a new OTP if the 5-minute window expires.

Publish

5) Publishing changes

  • Edits save as drafts.
  • Click Publish Live in the right sidebar to push updates.
  • Confirm publish; a success toast means it’s live.
Capabilities

6) What you can do in Bugence Visual Editor

  • Canvas overlay: highlight to edit, quick “Edit” badge, draft vs live control.
  • Text editing: formatting, links, undo/redo, meta stats.
  • Image editing: replace, preview, alt text quality meter.
  • Locks: Navbar/sidebar and flagged regions remain read-only.
  • Sidebar tools: Draft status, last publish, section count, Publish Live, Open Dashboard, View Live Page.
  • Live verification: Use View Live Page to confirm after publishing.
Tips

7) Quick tips

Alt text

Keep alt text concise (5–15 words) describing subject and context.

Quality check

Use the meta bar (words/chars/read time) before publishing.

Verify live

After publishing, open “View Live Page” and refresh to confirm changes.
