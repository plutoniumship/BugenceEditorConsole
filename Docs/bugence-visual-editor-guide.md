## 6) What you can do in Bugence Visual Editor
### Canvas mode (visual overlay)
- **Highlight to edit:** Hover to see outlines; double-click to open the editor.
- **Image editing:** Upload/replace, preview instantly, set alt text (with quality meter), save, then publish.
- **Text editing:** Rich text (bold/italic/headings/lists), links, undo/redo; meta bar shows words/chars/read time.
- **Lock awareness:** Navbar/sidebar and locked blocks stay read-only; edit buttons won‚Äôt open for them.
- **Draft vs live:** Draft changes stay local until you click **Publish Live**.
- **Quick badge:** Floating ‚ÄúEdit‚Äù badge appears near the highlighted block for single-click editing.

### Text editor popup
- Format text, add/remove links, tweak inline styles from the toolbar.
- See word/character counts and estimated read time before saving.
- Revert to the last saved version or to the previous revision if available.
- Save writes a draft to Bugence; publish pushes live.

### Image editor popup
- Upload new images (with size/type validation) and live preview.
- Edit alt text with a strength meter.
- Revert to the previous image/alt in one click.
- Save writes a draft to Bugence; publish pushes live.

### Sidebar (right panel)
- Shows draft status, last publish time, count of editable sections.
- **Publish Live** button to push all saved drafts.
- Links to **Open Dashboard** and **View Live Page** for quick verification.

### Live Preview / View Live Page
- Use **View Live Page** in the sidebar to open the live site in a new tab.
- After publishing, refresh the live tab to confirm changes.
